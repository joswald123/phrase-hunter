# phrase-hunter

JavaScript | HTML | CSS

### Description

Interactive word guessing game app. Using JavaScript & OOP (Object-Oriented Programming) to select a random, hidden phrase, 
which a player tries to guess, by clicking letters on an onscreen keyboard.
